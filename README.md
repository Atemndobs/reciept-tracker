## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npx expo start
   ```

3. Scan the QR code with:
   - iOS: Use the Camera app
   - Android: Use the Expo Go app

## Development Setup
